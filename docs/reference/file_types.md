# File Types

File types from extension, c("archive", "audio", "book", "code",
"config", data", "exec", "font", "image", "slide", "stats", text",
"video", "web")

## Usage

``` r
file_types
```

## Format

A data frame with 277 rows and 2 columns:

- ext:

  File extension

- type:

  Type of file
