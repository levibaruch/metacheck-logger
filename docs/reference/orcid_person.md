# Get Person Details for ORCiD

Get Person Details for ORCiD

## Usage

``` r
orcid_person(orcid)
```

## Arguments

- orcid:

  a vector of ORCiDs

## Value

A data frame of details

## Examples

``` r
# \donttest{
```
